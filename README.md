## Overview

**Anime-Ecstasy** is a modern web app for anime fans to:

- Browse trending anime with ratings, popularity, and more (powered by the [Jikan API](https://jikan.moe/))
- Add anime to a personal watchlist (like IMDb for anime)
- Search, filter, and sort your watchlist

> **Live Demo:** _Coming Soon_

---

## Features

- 🏆 Trending anime list with pagination
- ⭐ Add/remove anime to your local watchlist
- 🔍 Search and filter watchlist by genre and title
- 📊 Sort watchlist by rating
- 🎨 Responsive UI with [Tailwind CSS](https://tailwindcss.com/)
- ⚡ Fast development with [Vite](https://vitejs.dev/)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)
- [npm](https://www.npmjs.com/)

### Setup

1. **Clone your fork:**
   ```bash
   git clone https://github.com/<your-username>/Anime-Ecstasy.git
   cd Anime-Ecstasy
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The app will run at [http://localhost:5173](http://localhost:5173)

---

## Usage

- Browse trending anime on the home page
- Click the heart (💗) to add anime to your watchlist
- Access your watchlist from the navbar
- Search, filter by genre, and sort your watchlist

## Tech Stack

- **Frontend:** React, React Router
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
